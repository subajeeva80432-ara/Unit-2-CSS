You can clone the repository and open the HTML files to test the CSS.

All variables and reusable styles are organized for easy customization.

Mobile-first and responsive design principles are applied throughout.

Folder Structure (Suggested):

index.html → main HTML file

style.css → main CSS file

images/ → folder for images

icons/ → folder for SVG or other icons

components/ → optional folder for component-specific CSS
